# Maak een responsieve website met Bootstrap

Welkom in deze workshop.

Vandaag leren we hoe we een eenvoudige responsieve website opbouwen met HTML en Bootstrap.

Dit komt ongeveer overeen met de eerste helft van de eerste les van het vak ‘Responsive Design’ in het [graduaat Programmeren](https://www.thomasmore.be/opleidingen/graduaat/programmeren-dag-avondonderwijs/programmeren-dagonderwijs/antwerpen-0) aan de [Thomas More Hogeschool.](https://www.thomasmore.be/)
